# Historias de Usuario 

  1. Como atendente quero não poder registrar reservas quando o quarto ja estiver ocupado, porque evitaria problemas com reservas conflitantes.

  2. Como atendente quero que o sistema avise que um quarto esta reservado caso tente fazer outra reserva nele, para fazilitar o entendimento visau.

  3. Como atendente quero poder cadastrar os hospedes de uma reserva, para poder ter os dados necessario enquanto ele estiver hospedado.

  4. Como gerente quero ter um mapa de quartos, que mostre o tipo de suite e se esta ocupado, para fasilitar o gereciamento das reservas.

  5. Como recepcionista quero poder registrar itens de consumo a uma conta, para poder combrar tudo no checkout.

  6. Como gerente quero que o sistem diferencie os precos de suite, para ser cobrado o valor correto.

  7. Como funcionario da limpeza quero receber um aviso de quando um quarto precisa de limpeza, para poder saber qual quarto limpar

  8. Como hospeder quero poder pedir para que o meu quarto seje limpo, para ter uma limpeza no quarto quando necessaria

  9. Como funcionario da limpeza quero que o sistem funcione em dispositivos moveis, para eu poder telo em qualquer lugar do hotel.

  10. Como funcionario da limpeza quero poder registrar na conta do cliente o que ele consumiu do frigobar, para ter um controle do conumo.

  11. Como gerente quero definir itens diferentes para frigobares de suites de diferentes niveis, para ter um diferenciamento de items.

  12. Como gerente quero poder cadastrar e editar tipo de items(agua,chocolate,etc) para ter um controle de seus precos.

  13. Como gerente quero ter um controle de estoque, que atualize toda vez que algo for adicionado a um frigobar, ou consumido, para que ele sempre esteje certo

  14. Como gerente quero que poder registrar novos items ao estoque de uma vez, selecionando o tipo e quantidade, para ter otimizacao de tempo.

  15. Como gerernte quero que o sistema seje restrito a fucionario, para ser seguro.

  16. Como gerente quero ter um relatorio mensal de quanto foi ganho no mes, para saber quanto esta rendendo o hotel.

  17. Como gerente quero poder ter o registro dos dados dos hospeder por um 1 ano, para reaproveitalo caso ele volte.
